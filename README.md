# 🤖 Always-On Telegram Bot (Ready for Pella.app)

This is a live-tested, ready-to-deploy Telegram bot that runs 24/7 using [https://pella.app](https://pella.app)

---

## 📦 Features:
- Always on (no sleep)
- GitHub-ready
- Uses `pyTelegramBotAPI`
- Automatically starts on deploy

---

## 🚀 How to Use

### 1️⃣ Upload to GitHub
- Unzip this project
- Create a new public GitHub repository
- Upload all files

### 2️⃣ Deploy via Pella
- Go to https://pella.app
- Connect your GitHub
- Select this repo and click "Deploy"
- Add your Telegram bot token as env:
  - `BOT_TOKEN = xxxxxxxxxxxxxx`

✅ That’s it — the bot will auto-start!

Send `/start` to your bot to verify.
